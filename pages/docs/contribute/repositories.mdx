# Repository Overview

Pontus-X maintains several repositories for different aspects of the project. This chapter provides a comprehensive overview of these repositories, categorized by their purpose and functionality.

## Core Repositories

These repositories contain the essential codebases for the Pontus-X platform.

- **Pontus-X Ecosystem Organization:** [Pontus-X Ecosystem](https://github.com/Pontus-X)
- **Pontus-X Portal:** [MVG Portal](https://github.com/deltaDAO/mvg-portal)
- **Block Explorer:** [Oasis Protocol Explorer](https://github.com/oasisprotocol/explorer/)
- **Aquarius:** [Aquarius Repository](https://github.com/OceanProtocolEnterprise/aquarius)
- **Provider:** [Provider Repository](https://github.com/OceanProtocolEnterprise/provider)
- **Subgraph:** [Ocean Subgraph](https://github.com/OceanProtocolEnterprise/ocean-subgraph)
- **Contracts:** [Ocean Protocol Contracts](https://github.com/OceanProtocolEnterprise/contracts)

## Documentation Repositories

These repositories are dedicated to maintaining and updating the documentation for various components of the Pontus-X ecosystem.

- **Pontus-X Network Docs:** [Pontus-X Documentation](https://github.com/deltadao/pontusx-docs/)
- **Nautilus Docs:** [Nautilus Documentation](https://github.com/deltaDAO/nautilus/tree/main/docs)
- **Ocean Enterprise Docs:** [Ocean Enterprise Documentation](https://github.com/OceanProtocolEnterprise/docs)
- **Oasis Docs:** [Oasis Documentation](https://github.com/deltaDAO/oasis-docs)

## Tools and Libraries

These repositories provide additional tools and libraries that support the development and integration of the Pontus-X platform.

- **Nautilus:** [Nautilus Repository](https://github.com/deltaDAO/nautilus)
- **Explorer:** [Pontus-X Explorer](https://github.com/deltaDAO/pontusx-explorer)
