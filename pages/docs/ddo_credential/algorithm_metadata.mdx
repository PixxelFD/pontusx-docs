# Algorithm Metadata

| Attribute            | Type                                                           | Description                                                                                | Required           |
| -------------------- | -------------------------------------------------------------- | ------------------------------------------------------------------------------------------ | ------------------ |
| `container`          | [`container`](/docs/ddo_credential/container.mdx)                   | Object describing the Docker container image.                                              | :white_check_mark: |
| `language`           | `string`                                                       | Language used to implement the software.                                                   |                    |
| `version`            | `string`                                                       | Version of the software preferably in [SemVer](https://semver.org) notation. E.g. `1.0.0`. |                    |
| `consumerParameters` | [`consumerParameters`](/docs/ddo_credential/algorithm_consumer_parameters.mdx) | An object that defines required consumer input before running the algorithm                |                    |
