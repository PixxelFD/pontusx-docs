# Service

| Attribute               | Type                                                                    | Description                                                                                                                                  | Required                                                         |
| ----------------------- | ----------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- |
| `id`                    | `string`                                                                | Unique ID                                                                                                                                    | :white_check_mark:                                               |
| `type`                  | `string` enum                                                           | Type of service `access`, `compute`                                                                                                          | :white_check_mark:                                               |
| name                    | `string`                                                                | Service friendly name                                                                                                                        | :white_check_mark:                                               |
| displayName             | [`language value object`](/docs/ddo_credential/language_i18n)                | Descriptive name of the service used for human readability in applications.                                                                  |                                                                  |
| `description`           | [`language value object`](/docs/ddo_credential/language_i18n)                | Service description                                                                                                                          |                                                                  |
| `datatokenAddress`      | `string`                                                                | Smart Contract address of the service specific token                                                                                         | :white_check_mark:                                               |
| `serviceEndpoint`       | `string ` (URL)                                                         | Access Controller(Provider) URL                                                                                                              | :white_check_mark:                                               |
| `files`                 | `string` (encrypted [`files object`](/docs/ddo_credential/files_object.mdx)) | Encrypted access control information of the files (encrypted by the access controller(provider))                                             | :white_check_mark:                                               |
| `timeout`               | `number`                                                                | Describing how long the service can be used after consumption is initiated. A timeout of `0` represents no time limit. Expressed in seconds. | :white_check_mark:                                               |
| `compute`               | [`compute_object`](/docs/ddo_credential/compute_object)                      | If service is of `type` `compute`, holds information about the compute-related privacy settings & resources.                                 | :ballot_box_with_check: (Required for `type` `compute` services) |
| `consumerParameters`    | [`consumerParameters`](/docs/ddo_credential/service_consumer_parameters)     | An object the defines required consumer input before consuming the asset.                                                                    |                                                                  |
| `additionalInformation` | Object                                                                  | Stores additional information, this is customizable by publisher                                                                             |                                                                  |
| `state`                 | `number` enum [`state`](/docs/ddo_credential/lifecycle_states)               |                                                                                                                                              | :white_check_mark:                                               |
| `credentials`           | [`credentials`](/docs/ddo_credential/credentials)                            | Access Control Credentials                                                                                                                   | :white_check_mark:                                               |
| `dataSchema`            | [`remoteObject`](/docs/ddo_credential/remote_object)                         | A remote document defining the schema of a data service                                                                                      | (only `dataset` type assets)                                     |
| `inputSchema`           | [`remoteObject`](/docs/ddo_credential/remote_object)                         | A remote document defining the expected schema of the input data for a specific algorithm                                                    | (only `algorithm` type assets)                                   |
| `outputSchema`          | [`remoteObject`](/docs/ddo_credential/remote_object)                         | A remote document defining the schema of the algorithm output                                                                                | (only `algorithm` type assets)                                   |
