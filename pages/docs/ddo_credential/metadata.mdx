# metadata

| Attribute               | Type                                                           | Description                                                                                                                                      | Required                                                      |
| ----------------------- | -------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------- |
| `created`               | `ISO date/time string`                                         | Contains the date of the creation of the dataset content in ISO 8601 format preferably with timezone designators, e.g. `2000-10-31T01:30:00Z`.   | :white_check_mark:                                            |
| `updated`               | `ISO date/time string`                                         | Contains the date of the creation of the dataset content in ISO 8601 format preferably with timezone designators, e.g. `2000-10-31T01:30:00Z`.   | :white_check_mark:                                            |
| `description`           | [`language value object`](/docs/ddo_credential/language_i18n)                                        | Details of what the resource is. For a dataset, this attribute explains what the data represents and what it can be used for.                    | :white_check_mark:                                            |
| `copyrightHolder`       | `string`                                                       | The party holding the legal copyright.                                                                                                           |                                                               |
| `name`                  | `string`                                                       | Descriptive name of the asset.                                                                                                                   | :white_check_mark:                                            |
| `displayTitle`          | [`language value object`](/docs/ddo_credential/language_i18n)                                        | Descriptive name of the asset used for human readability in applications.                                                                        |                                                               |
| `type`                  | `string` enum                                                  | Asset type. Includes `"dataset"` (e.g. csv file), `"algorithm"` (e.g. Python script). Each type needs a different subset of metadata attributes. | :white_check_mark:                                            |
| `author`                | `string`                                                       | Name of the entity generating this data (e.g. `deltaDAO AG`).                                                                                    |                                                               |
| `providedBy`            | `string` URL (participant credential)                          | Verifiable Credential of the entity offering this asset.                                                                                         | :white_check_mark:                                            |
| `license`               | [`licenseObject`](/docs/ddo_credential/license_object.mdx)          | Licenses applied for all services of this asset.                                                                                                 |                                                               |
| `links`                 | Object                                                         | Mapping of URL titles(key) and URL strings(value)                                                                                                |                                                               |
| `attachments`          | Array of [`remoteObject`](/docs/ddo_credential/remote_object)                                        | List of remote files attached to the asset                                                                                                       |                                                               |
| `tags`                  | Array of `string`                                              | Array of keywords or tags used to describe this content. Empty by default.                                                                       |                                                               |
| `categories`            | Array of `string`                                              | Array of categories associated to the asset.                                                                                                     |                                                               |
| `additionalInformation` | Object                                                         | Stores additional information, this is customizable by publisher                                                                                 |                                                               |
| `algorithm`             | [`Algorithm Metadata`](/docs/ddo_credential/algorithm_metadata.mdx) | Information about asset of type `algorithm`                                                                                                      | :ballot_box_with_check: (Required for asset type `algorithm`) |
